**Storyline Quiz Parser — Description**

This tool is a browser-based utility designed to extract and analyze quiz data from Articulate Storyline published outputs. By uploading a ZIP file containing the Storyline HTML5 output (including `data.js` and slide JavaScript files), the application automatically parses and reconstructs quiz content.

It identifies question types such as Multiple Choice, True/False, and Drag-and-Drop, and extracts key elements including question text, answer options, correct answers, and scoring information.

An optional Answer Key feature allows users to paste expected answers and instantly validate them against the parsed results, highlighting matches and mismatches for quick review.

The interface supports both table and card views, along with search, filtering, and CSV export functionality for easy analysis and reporting.

Built entirely with client-side JavaScript, the tool runs locally in the browser with no server dependency, ensuring speed, privacy, and ease of use.
